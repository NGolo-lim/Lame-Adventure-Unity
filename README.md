2D Platformer 

Платформа: PC / Unity 2D (C#)
Жанр: Puzzle, Action

Описание
Хардкорный платформер из 4 уровней, где игроку нужно собирать золото, избегать патрулей и турелей, чтобы добраться до финиша. 


Особенности:
* Динамическая система сбора предметов и открытия дверей.
* Реализовано ИИ врагов (патрулирование, зрение).
* Собственный UI, система переходов между уровнями и звуковое сопровождение.

Стек технологий:
* Unity 2D (Physics 2D, Raycasting).
* C# (Singleton pattern, Coroutines, Scene Management).
